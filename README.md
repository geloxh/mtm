### mtm (My Task Manager)
Maintainable and scalable application and web app task manager project for professional documentation and leveraging productivity accomplishments.

### Project Structure

```
mtm/
├── apps/
│   ├── web/          # Next.js
│   ├── api/          # Express
│   ├── mobile/       # Expo (React Native)
│   └── desktop/      # Tauri
├── packages/
│   └── ui/           # Shared components
├── package.json
└── turbo.json
```
